# vanilla-php-api
Simple API using vanilla PHP and PDO Extension

## How to use
### Use your connection local or remote
```php
private $host = 'name_or_ip_remote_server'; 
private $user = 'your_user';
private $pass = 'your_password';
private $port = '3306';
```
